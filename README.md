# Hillshire Brands (hillshire-brands)
Hillshire Brands was a U.S.-based food company that produced and sold meat-centric food solutions including Jimmy Dean, Ball Park, Hillshire Farm, State Fair, and Sara Lee branded products. The company was acquired by Tyson Foods in 2014 and its operations have been integrated into the Tyson Foods organization. Hillshire Brands does not currently publish a public developer API or API documentation.
